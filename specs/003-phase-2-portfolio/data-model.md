**On-disk**: `${DEEP_FINANCE_DATA_DIR}/{universe_name}.parquet`, schema per

## E2 — `PortfolioWeights`

A daily long-only weight vector for one (method, universe) pair. Lives in
memory only — recomputed per session by `pages/2_💼_Portfolio.py` or
`scripts/run_backtests.py --portfolio`.

**In-memory representation**: pandas DataFrame indexed by date, with one
column per asset (length N). Each row sums to 1 ± 1e-6, every entry in
[0, 1].

**Validation rules**:
- `weights.sum(axis=1).between(1 - 1e-6, 1 + 1e-6).all()` — simplex
- `(weights >= 0).all().all()` — long-only
- Index is `is_monotonic_increasing` and matches the universe's date range

## E4 — `PortfolioBacktestPanel`

Long-format DataFrame at `data/backtests/portfolio_results.parquet` —
drives Tab 4. Pre-computed offline by `scripts/run_backtests.py --portfolio`;
read by the page via `@st.cache_data`.

**Fields**:

| Field | Type | Required | Notes |
|---|---|---|---|
| `date` | `date32[day]` | ✅ | Trading day |
| `universe` | `string` | ✅ | `"etfs"` or `"20stock"` |
| `method` | `string` | ✅ | One of: `"equal_weight"`, `"min_variance"`, `"max_diversification"`, `"diversity_weighted"`, `"deep_portfolio"`, `"alloc_25_25_25_25"`, `"alloc_50_10_20_20"`, `"alloc_10_50_20_20"`, `"alloc_40_40_10_10"` (last 4 only for `etfs`) |
| `vol_scaling` | `bool` | ✅ | `True` → portfolio return rescaled to σ_target = 0.10 annualized |
| `cost_rate` | `float64` | ✅ | `0.0001` or `0.0010` (= 0.01% / 0.10%) |
| `portfolio_return` | `float64` | ✅ | Daily portfolio P&L net of turnover cost |

**Row cardinality**: roughly:
- etfs: 9 methods × 2 vol-scaling × 2 cost-rates × ~3900 trading days
  ≈ 140 k rows
- 20stock: 5 methods × 2 × 2 × ~5000 days ≈ 100 k rows
- Total ≈ 240 k rows, well under 50 MB parquet

**Validation rules**:
- `set(universe.unique()) == {"etfs", "20stock"}`
- `set(vol_scaling.unique()) == {True, False}`
- `set(cost_rate.unique()) == {0.0001, 0.0010}`
- For each (universe, method, vol_scaling, cost_rate) tuple, dates are
  monotonic increasing and non-empty
- ETF-only methods (alloc_*) MUST NOT appear with `universe == "20stock"`

**State transitions**: regenerated by `scripts/run_backtests.py --portfolio`
whenever input parquets refresh or checkpoints retrain.
